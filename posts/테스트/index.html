📄 Hugo 날짜 포맷 관련 대화 (Markdown)

User:
그러면 2025.07 이렇게 하려면?

Assistant:
2025.07 형식으로 날짜를 출력하려면 Hugo에서 이렇게 쓰면 됩니다:

{{ .Date.Format "2006.01" }}

✅ 설명

  • 2006 → 연도 (4자리)
  • 01 → 월 (2자리 숫자, 앞에 0 포함)
  • . → 그냥 구분자

User. 마크다운으로 출력해 줘

Assistence:

📅 예시 결과

날짜 출력
2025년 7월 22일 2025.07
2024년 12월 5일 2024.12
2023년 3월 9일 2023.03

필요하면 뒤에 일(day)까지 붙여서 2025.07.22처럼 만들 수도 있고,
슬래시(/)나 하이픈(-)도 쓸 수 있어:

  • {{ .Date.Format "2006-01-02" }} → 2025-07-22
  • {{ .Date.Format "2006/01" }} → 2025/07

User:
2025/07/21 형태라면?

Assistant:
2025/07/21 형태로 날짜를 출력하려면 Hugo에서 아래처럼 쓰면 됩니다:

{{ .Date.Format "2006/01/02" }}

🧩 포맷 구성

포맷 요소 의미 결과 예시
2006 연도 2025
01 월 (2자리) 07
02 일 (2자리) 21
/ 구분자 /

✅ 전체 예시

{{ .Date.Format "2006/01/02" }}

결과 → 2025/07/21

필요하면 다른 형식 (YY.MM.DD, 7월 21일, July 21, 2025) 등도 쉽게 만들어줄 수 있어.
원하는 형식 있으면 말해줘!
